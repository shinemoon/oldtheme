这个，非常的直观，也没什么好多说的——不过却真的很好用。 Add to search bar

你可以在任意搜索的地方右键，然后选择Add to search bar-> 于是我的搜索栏里有了土豆、wiki、Firefox addon、百度mp3、某些论坛的搜索栏等等等等。

不能不装的好插件啊。
